Really starting

This app is an open service developed by the Galaxy and CloudBioLinux communities, and modified for use for Bioconductor. The goal is to make it easy to get started doing scalable biological analysis on cloud through Bioconductor resorces. The open source code is available on Github allowing you to also run this application locally.

Launching servers on the Amazon cloud will incur usage fees from user's Amazon resources. By using this service you acknowledge your sole responsibility for any costs accrued.
